## ‚ö° How to run

Clone the repository

```bash
  git clone https://github.com/meeerrrm/laravel-blog-with-seo.git
```
Start
```bash
  cd src/BLOGSYS
  composer install
  npm install

  copy .env.example .env
  php artisan key:generate
  
  php artisan serve
```
